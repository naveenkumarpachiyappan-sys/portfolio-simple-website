Personal Portfolio Website

This is a responsive 2-page portfolio website created using HTML5 and CSS3. The website is designed to introduce myself, showcase my skills and projects, and provide a contact page for visitors.

Features
Clean and modern UI
Responsive layout
Home page with introduction
Contact page with a contact form
Semantic HTML structure
Organized CSS using the BEM naming convention
Technologies Used
HTML5
CSS3
Google Fonts
Pages
Home – Displays my introduction, skills, and projects.
Contact – Contains a contact form and links to my social media profiles.
Project Purpose

This project was built as part of my web development learning journey to practice HTML and CSS fundamentals, responsive design, and webpage structuring.
